# Swiss MSE Module Nodes — Suggested Additions

Based on the official Swiss MSE (Master of Science in Engineering) programme structure and profile pages.

## Programme-wide module structure

- `mse-technical-scientific-modules` -> `root`  
  Connection: `related`  
  Notes: Profile-specific theory modules (TSM).

- `mse-fundamental-theoretical-principles` -> `mathematics`  
  Connection: `bridge`  
  Notes: Shared advanced theory block (FTP) spanning higher mathematics, physics, information theory, chemistry.

- `mse-context-modules` -> `bridge`  
  Connection: `bridge`  
  Notes: Shared context block (CM) covering management, communication, law, and project work.
